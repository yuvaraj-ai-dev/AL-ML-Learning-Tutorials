
# Image & Video Processing Applications in Python

This course contains module-level Jupyter notebooks (one per module) plus a capstone project.

**Structure (one notebook per module):**
- 00-Setup-and-Intro.ipynb
- 01-Image-Basics-I_O-and-Visualization.ipynb
- 02-Pointwise-Operations-and-Histograms.ipynb
- 03-Filtering-and-Enhancement.ipynb
- 04-Geometry-and-Transforms.ipynb
- 05-Edge-and-Corner-Detection.ipynb
- 06-Feature-Detection-and-Descriptors.ipynb
- 07-Image-Segmentation.ipynb
- 08-Object-Detection-and-Classical-Methods.ipynb
- 09-Video-Processing-Basics.ipynb
- 10-Motion-Estimation-and-Tracking.ipynb
- 11-Deep-Learning-for-Vision-Basics.ipynb
- 12-Advanced-Video-Applications.ipynb
- CAPSTONE-Project-Object-Detection-and-Tracking-Pipeline.ipynb

## How to use
1. Create a virtual environment and install requirements from `requirements.txt`.

2. Launch Jupyter and open notebooks.

3. Each notebook contains runnable examples and exercises.

## Sample data

Small synthetic sample images and a short demo video are included under `datasets/`.
